relying on a programmatic approach to create your first user—often called "database seeding"—is a standard industry practice because it is repeatable and does not require external graphical tools.

We will create a quick, one-off Python script to generate your first admin user directly through your Docker container.

Here is how to do it step-by-step:

Step 1: Create the Seeding Script
In your project root directory (alongside main.py), create a new file named create_admin.py and paste the following code into it:

Python
from database import SessionLocal
import models
import auth

def create_superuser():
    db = SessionLocal()
    try:
        # Check if an admin already exists to prevent duplicates
        existing_admin = db.query(models.Admin).filter(models.Admin.username == "admin").first()
        if existing_admin:
            print("Admin user already exists!")
            return

        # Hash the password and create the user
        hashed_pw = auth.get_password_hash("admin123")
        new_admin = models.Admin(username="admin", hashed_password=hashed_pw)
        
        db.add(new_admin)
        db.commit()
        print("✅ Success! Admin user created.")
        print("Username: admin")
        print("Password: admin123")
        
    except Exception as e:
        print(f"❌ Error creating admin: {e}")
    finally:
        db.close()

if __name__ == "__main__":
    create_superuser()
Step 2: Rebuild Your Containers
Because you added a new Python file to your local directory, you need to rebuild your API image so Docker copies the new create_admin.py file into the container.

Run this in your terminal:

Bash
docker compose up -d --build
Step 3: Execute the Script Inside Docker
Once the containers are back up and running, you can tell Docker to execute that specific file inside the running api container.

Run this command:

Bash
docker compose exec api python create_admin.py
You should see the success message printed in your terminal.

Step 4: Log In
You can now navigate to http://localhost:8000/admin in your web browser.

Use the credentials you just generated:

Username: admin

Password: admin123
